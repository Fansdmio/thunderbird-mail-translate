# Thunderbird 邮件翻译

阅读收到的邮件时，一键将**主题 + 正文**全文替换为译文，并支持原文/译文切换。

## 功能

- 手动点击工具栏按钮翻译当前邮件
- 全文替换正文（尽量保留 HTML 结构）与主题显示
- 一键切换 原文 ⇄ 译文（不修改服务器上的原邮件）
- 默认 **微软翻译（免配置）**
- 可选 **Azure 官方** 或 **自定义 OpenAI 兼容 AI**（端点 / API Key / 模型 ID）
- 目标语言在设置页全局选择

## 安装（开发加载）

1. 打开 Thunderbird
2. 菜单 → 设置 → 附加组件与主题 → 齿轮图标 → **调试附加组件**
3. 点击 **临时加载附加组件**
4. 选择本仓库中的 `manifest.json`

## 使用

1. 打开一封收到的邮件
2. 点击阅读窗工具栏 **翻译**
3. 翻译完成后按钮变为 **显示原文**，可来回切换

## 设置

附加组件管理页 → 邮件翻译 → 选项：

| 项 | 说明 |
|----|------|
| 翻译引擎 | 微软免配置 / Azure / 自定义 AI |
| 目标语言 | 如简体中文 |
| Azure | 订阅密钥、区域 |
| 自定义 AI | Base URL、API Key、模型 ID |
| 测试连接 | 用短句验证引擎是否可用 |

## 开发

```powershell
npm install
npm test
```

## 隐私

- 邮件内容仅发送到你选择的翻译服务（微软或自填 AI 端点）
- API Key 仅保存在本机 `storage.local`
- 本扩展不经过作者自建中继服务器

## 限制

- 微软免 Key 接口为非官方端点，可能变更或限流
- 首版不支持写信翻译、自动翻译、对照模式
- 列表中的主题不会被改写

## 兼容

- Thunderbird 128+（MailExtensions）

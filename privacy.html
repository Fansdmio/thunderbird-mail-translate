Mail Translate Privacy Policy

Extension ID: mail-translate@uuyo.pw · Effective: 2026-07-19 · Version: 0.1.7

Summary

  • Mail content is sent only to the translation service you select.
  • API keys and settings are stored only on this device via Thunderbird local storage.
  • This add-on does not operate a relay/proxy server and does not collect analytics.

Data processed

  • Message subject and body of the mail you manually choose to translate.
  • Settings: engine choice, target language, optional Azure key/region, optional AI Base URL / API key / model.

Where data is sent

  • Microsoft Translator (default, no key): temporary token from edge.microsoft.com, translation via Microsoft Translator Cognitive endpoints.
  • Microsoft Azure: your Azure Translator endpoint with the key/region you provide.
  • Custom AI: only the Base URL host you configure (OpenAI-compatible). Host permission is requested at save/test time.

Nothing is sent until you click Translate (or Test connection in settings).

Permissions

  • messagesRead / messagesModify: read displayed mail and replace body in the reading pane (local UI only; server mail is not rewritten).
  • storage: save settings.
  • tabs: associate translation state with the reading tab.
  • Host permissions for Microsoft Translator endpoints; optional host permission for your custom AI origin.

Retention

In-session translation cache lives in memory and is discarded when the tab closes or Thunderbird restarts. Provider-side retention follows each third-party service’s own policy.

Contact

For privacy questions about this add-on, open an issue on the project repository or contact the developer listed on addons.thunderbird.net.

邮件翻译 隐私政策

扩展 ID：mail-translate@uuyo.pw · 生效日期：2026-07-19 · 版本：0.1.7

摘要

  • 邮件内容仅发送到你选择的翻译服务。
  • API 密钥与设置仅保存在本机（Thunderbird 本地存储）。
  • 本扩展不经过作者中继服务器，也不收集分析数据。

处理的数据

  • 你手动点击翻译时的邮件主题与正文。
  • 设置项：引擎、目标语言，以及可选的 Azure / 自定义 AI 配置。

数据发送目标

  • 微软翻译（默认免配置）：令牌来自 edge.microsoft.com，译文来自微软 Translator 接口。
  • 微软 Azure：使用你填写的密钥与区域调用 Azure Translator。
  • 自定义 AI：仅请求你配置的 Base URL 主机；保存/测试时按需申请主机权限。

在你点击「翻译」或设置页「测试连接」之前，不会发送邮件内容。

权限说明

  • messagesRead / messagesModify：读取当前邮件并在阅读窗替换显示（不修改服务器上的原邮件）。
  • storage：保存设置。
  • tabs：关联标签页翻译状态。
  • 微软翻译主机权限；自定义 AI 的可选主机权限。

保留期限

会话内译文缓存在内存中，关闭标签或重启 Thunderbird 后清除。第三方服务侧保留策略以其自身政策为准。

联系

隐私相关问题可通过项目仓库 Issue，或 ATN 商店页上的开发者联系方式反馈。
